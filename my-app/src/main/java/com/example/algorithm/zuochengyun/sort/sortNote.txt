排序算法的稳定性: 相同的元素排完序之后,相对次序保持不变。

基础类型排序 和稳定性没关系。

稳定性算法:  冒泡排序, 插入排序, 归并排序，桶排序(计数排序,基数排序)
非稳定性排序: 选择排序，快排，堆排序

N*logN 排序有三种:   归并排序, 快排，堆排序


排序算法    时间复杂度   空间复杂度   稳定性
选择排序    O(N平方)     O(1)       否
冒泡        O(N平方)     O(1)      可以
插入        O(N平方)     O(1)      可以
归并        O(N*logN)   O(N)       可以
快排        O(N*logN)   O(logN)    否
堆排序      O(N*logN)    O(1)       否

基于比较的排序, 目前没有一种算法 时间复杂度   O(N*logN),额外空间复杂度O(1),又稳定的排序。
